# Example Push Guide

This file demonstrates a successful push to the repository.

## Instructions to Replicate

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/b4130478-web/iyf-s10-week-5-b4130478-web.git
   cd iyf-s10-week-5-b4130478-web
   ```

2. **Make a change** (edit any file or create a new one):
   ```bash
   echo "My changes" >> README.md
   ```

3. **Stage your changes**:
   ```bash
   git add .
   ```

4. **Commit your changes**:
   ```bash
   git commit -m "Add example changes"
   ```

5. **Push to the remote**:
   ```bash
   git push origin main
   ```

## Done! ✓

Your changes are now pushed to GitHub!
